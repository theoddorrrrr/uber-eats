# Goods-Shop

## Run :

```.bash
    git clone https://github.com/theoddorrrrr/uber-eats.git

    cd uber-eats

    npm install
```

### After initialization the project - run:

```.bash
    npm run start
```

### for development or

```.bash
    npm run build
```

### to get (dist) folder, which containts all minificated files to run site.
